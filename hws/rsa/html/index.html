Main Page

This documentation describes three classes provided in the skeleton code for the RSA homework.

  • CipherText: a class to hold the cipher text for any RSA encrypted data
  • RSAKey: a class to hold a public or private (or both) key
  • RSA: the main class that implementes RSA. There are six methods within that need to be completed for this assignment: RSA.generateKeys(), RSA.encrypt(), RSA.decrypt(), RSA.sign(), RSA.checkSign(), and RSA.crack().

Lastly, the command line parameters are specified at cmdparam.
